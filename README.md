To rebuild the project from scratch:

```bash

cd docs
# Read docs/PRD.md → docs/IMPLEMENTATION_PLAN.md → start building
```

Or to view the full source:

```bash
git clone https://github.com/littlebom/U-Course-Design-Edx.git
```
